CSS基础

1.CSS简介

1.1 概念

CSS（层叠样式表）是一种标记语言，用于设置HTML页面中的文本内容（字体、大小、对齐方式等）、图片的外形（宽高、边框样式、边距等）及版面的布局和外观显示样式。

1.2 CSS语法规范

选择器 {样式}

<style>
    p {
        color: red;
        font-size: 12px;
    }
</style>

2.CSS基础选择器

基础选择器由单个选择器组成，包括标签选择器、类选择器、id选择器和通配符选择器。

1. 标签选择器：可以选出所有相同的标签

为同类型的标签统一设置样式，谁需要谁调用。

   <style>
       p {
           color: green;
       }
       div {
            color: pink;
       }
   </style>

2. 类选择器：差异化选择不同的标签

样式点定义，结构类调用；一个或多个，开发最常用。

   <head>
       <style>
       .red {
           color: red;
       }
       .star-sing {
           color: green;
       }
       .memeda {  
           color: pink;
       }
       </style>
   </head>
   <body>
       <ul>
           <li class="red">冰雨</li>
           <li class="red">来生缘</li>
           <li>李香兰</li>
           <li class="memeda">生僻字</li>
           <li class="star-sing">江南style</li>
       </ul>
       <div class="red">我也想变红色</div>
   </body>

多类名：同一个div可调用多个class

   <head>
       <style>
           .red {
               color:  red;
           }
           .font35 {
               font-size: 35px;
           }
       </style>
   </head>
   <body>
       <div class="red font35">刘德华</div>
   </body>

3. id选择器

HTML元素以id属性来设置id选择器，CSS中id选择器以#来定义。id只可以被调用一次，class可以被调用多次

    <head>
        <style>
            #nav(id名) {；
                color: red;
                ···
            }
        </style>
    </head>
    <body>
        <div id="nav"></div>
    </body>

4. 通配符选择器

通配符选择器使用*定义，表示选取页面中所有（例如HTML、div、li等）元素（标签）

通配符选择器不需要调用，自动就给所有的元素使用样式

    <style>
        * {
            color: red;
        }
    </style>

5. id选择器和类选择器的区别

类选择器（class）类比人的名字，一个人可以有多个名字，同时一个名字也可以被多个人使用

id选择器类比人的身份证号码，全中国是唯一的，不得重复

3.CSS字体属性

CSS Fonts（字体）属性用于定义字体系列、大小、粗细和文字样式（如斜体）

1. 字体系列

字体之间使用逗号隔开，若是多个单词则需要加引号，可以直接给body定义

```
<html>
    <style>
    body {
        font-family: 'Times New Roman', Times, serif;
    }
    h2 {
        font-family: '微软雅黑';
    }
    p {
        font-family: 'Times New Roman', Times, serif;
    }
    </style>
</head>
<body>
    <h2>pink的秘密</h2>
    <p>那一抹众人中最漂亮的颜色,</p>
    <p>优雅,淡然,又那么心中清澈.</p>
    <p>前端总是伴随着困难和犯错,</p>
    <p>静心,坦然,攻克一个又一个.</p>
    <p>拼死也要克服它,</p>
    <p>这是pink的秘密也是老师最终的嘱托.</p>
</body>
```

2. 字体大小

CSS使用font-size属性定义字体大小，可以直接给body指定整个页面文字的大小
p { font-size: 20px; }
注：标题标签需要单独指定文字大小

3. 字体粗细

CSS使用font-weight属性定义字体粗细

```
<head>
    <style>
    .bold {
        font-weight: 700;    
    }
    h2 {
        font-weight: 400;   
    }
    </style>
</head>
<body>
    <h2>pink的秘密</h2>
    <p class="bold">拼死也要克服它</p>
</body>
```

4. 字体样式（风格）

CSS使用font-style属性定义字体样式

```
<html>
    <style>
    p {
        font-style: italic;
    }
    em {
        font-style: normal;
    }
    </style>
</head>
<body>
    <p>同学,上课时候的你</p>
    <em>下课时候的你</em>
</body>
```

5. 复合属性

字体属性可以综合来写，格式顺序不可改变

格式：font: font-style font-weight font-size/line-height font-family;

font-size和font-family属性必须有，其余属性可以省略

```
<head>
    <style>
    /* 想要div文字变倾斜 加粗 字号设置为16像素 并且 是微软雅黑 */
    div {
        /* font-style: italic;
        font-weight: 700;
        font-size: 16px;
        font-family: 'Microsoft yahei'; */
        /* 复合属性: 简写的方式  节约代码 */
        /* font: font-style  font-weight  font-size/line-height  font-family; */
        /* font: italic 700 16px 'Microsoft yahei'; */
        font: 20px '黑体';
    }
    </style>
</head>
<body>
<div>三生三世十里桃花,一心一意百行代码</div>
</body>
```

6. 字体属性总结

属性 表示 注意点
font-size 字号 单位是px
font-family 字体 实际工作中按照团队约定书写字体
font-weight 字体粗细 加粗是700或bold，不加粗是400或normal；数字后不跟单位
font-style 字体样式 倾斜是italic，不倾斜是normal；常用normal
font 字体连写 1.字体连写是有顺序的，不能随意换位置 2.其中字号和字体必须同时出现

4.CSS文本属性

CSS Text（文本）属性可定义文本的外观，例如文本的颜色、对齐文本、装饰文本、文本缩进、行间距等

4.1 文本颜色

color属性用于定义文本的颜色
div { color: red; }

4.2 对齐文本

text-align属性用于设置元素内文本内容的水平对齐方式

h1 {
    text-align: center/left/right;
}

4.3 装饰文本

text-decoration属性规定添加到文本的修饰，可以给文本添加下划线、删除线、上划线等

div {
text-decoration:underline/none;
}

4.4 文本缩进

text-indent属性常用于段落的首行缩进

p {
    text-indent: 2em;
}

4.5 行间距

text-height属性用于设置行间距（行高），行高=上间距+文本高度+下间距

p {
    text-height: 25px;
}

4.6 文本属性总结
|属性|表示|注意点|
|–|–|–|
|color|文本颜色|通常使用十六进制的简写形式|
|text-align|文本对齐|left/center/right|
|text-decoration|文本装饰|添加下划线：underline；取消下划线：none|
|text-indent|文本缩进|通常text-indent: 2em|
|text-height|行高|行高 = 上间距 + 文本高度 + 下间距|

5.CSS的引入方式

  1. 行内样式表（行内式）

    行内样式表（内联样式表）是在元素标签内部的<style>属性中设定CSS样式，适合于修改简单样式

    <style>其实就是标签的属性，将style直接放在<div>标签中

    控制当前标签的样式设置

    <div style="color: red; font-size: 12px;">青春不常在，抓紧去学习</div>

  2. 内部样式表（嵌入式）

    内部样式表（内嵌样式表）是写到HTML页面内部，是将所有的CSS代码抽取出来，单独放到一个<style>标签中

    一般将<style>标签放在<head>标签中

    控制当前页面的元素样式设置

    <style>
    div {
        color: red;
        font-size: 12px;
    }
</style>

  3. 外部样式表（链接式）

    外部样式表是样式单独写到CSS文件中，之后把CSS文件引入到HTML页面中使用

    新建后缀为.css的文件，把所有CSS代码放入其中；在HTML页面中，使用<link>标签引入这个文件；<link>标签放在<head>标签中

    控制多个页面的元素样式设置

    
<link rel="stylesheet" href="css文件路径/style.css">

6.CSS的复合选择器

常用的复合选择器包括：后代选择器、子选择器、并集选择器、伪类选择器

  1. 后代选择器

    后代选择器（包含选择器）可以选择父元素里面子元素

    元素1 元素2 {样式声明}

    注：上述语法表示选择元素1里面的所有元素2（后代元素）

  2. 子选择器

    子元素选择器只能选择作为某元素的最近以及子元素，简单理解就是选亲儿子元素

    元素1 > 元素2 {样式声明}

    注：上述语法表示选择元素1里面的所有直接后代元素2（子元素）

  3. 并集选择器

    并集选择器可以选择多组标签，同时为他们定义相同的样式，常用于集体声明

    元素1 , 元素2 {样式声明}

    注：上述语法表示元素1和元素2

  4. 伪类选择器之链接伪类选择器

    伪类选择器用于向某些选择器添加特殊的效果，比如给链接添加特殊效果

    a:link/visited/hover/active
    （顺序不可改变）

    a链接需要单独指定样式

     a {
     color: gray;
 }
 a:hover {
     color: red;
 }
    选项 解释
    link 选择所有未被访问的链接
    visited 选择所有已被访问的链接
    hover 选择鼠标指针位于其上的链接
    active 选择活动链接（鼠标按下未弹起的链接）

    注：上述语法表示元素1和元素2

  5. 伪类选择器之：focus伪类选择器

    ：focus伪类选择器用于选取获得焦点的表单元素

    input:focus {
 background-color: yellow;
}

  6. 复合选择器总结

    选择器 作用 特征 使用情况 隔开符号及用法
    后代选择器 用来选择后代元素 可以是子孙后代 较多 符号是空格 .nav a
    子代选择器 选择最近一级元素 只选亲儿子 较少 符号是大于 .nav>p
    并集选择器 选择某些相同样式的元素 可以用于集体声明 较多 符号是逗号 .nav，.header
    链接伪类选择器 选择不同状态的链接 跟链接相关 较多 符号是大括号 a{}和a:hover{}
    ：focus伪类选择器 选择获得光标的表单 跟表单相关 较少 符号是冒号 input:focus
    ## 8.CSS的元素显示模式

元素显示模式就是元素（标签）以什么方式进行显示

  1. 元素显示模式分类

    HTML分为块元素、行内元素和行内块元素

    元素模式 元素排列 设置样式 默认宽度 包含 注意
    块级元素 一行只能放一个块级元素 可以设置宽度高度 容器的100% 容器级可以包含任何标签 文字类的元素内不能放块级元素
    行内元素 一行可以放多个行内元素 不可以直接设置宽度高度 它本身内容的宽度 容纳文本或者其他行内元素，不可放块元素 链接内可以放块元素
    行内块元素 一行放多个行内块元素 可以设置宽度高度 它本身内容的宽度

  2. 元素显示模式转换

    一个模式的元素需要另外一种模式的特性

    转换为块元素display:block;
    转换为行内元素display:inline;
    转换为行内块元素display:inline-block;
    注：单行文字垂直居中（CSS没有提供代码）line-height：让文字的行高等于盒子的高度

9.CSS的背景

背景属性可以设置背景颜色、背景图片、背景平铺、背景图像固定、背景图片位置等

  1. 基础写法
    作用 属性 值
    背景颜色 background-color 十六进制数/透明（transparent）
    背景图片 background-image url(图片路径)
    背景平铺 background-repeat no-repeat/repeat/repeat-x/repeat-y
    背景图像固定 background-attachment fixed/scroll
    背景图片位置 background-position center 20px
  2. 复合写法
    background: transparent url(image.jpg) repeat-y fixed top;
  3. 背景颜色半透明
    background: rgba(0, 0, 0, .3);

10.CSS的三大特性

CSS的三个特性：层叠性、继承性、优先级

  1. 层叠性

    代码是逐条执行，若有冲突则会覆盖之前样式

  2. 继承性

    行高1.5这样写法最大的优势就是里面子元素可以根据自己文字大小自动调整行高

    继承时行高是当前元素的文字大小的1.5倍

    font:12px/1.5 Microsoft YaHei

  3. 优先级

    复合选择器的权重会叠加，但权重叠加不进位

    选择器 选择器权重
    继承 或者 * 0，0，0，0
    元素选择器 0，0，0，1
    类选择器，伪类选择器 0，0，1，0
    ID选择器 0，1，0，0
    行内样式 style=”” 1，0，0，0
    !important 重要的 ∞ 无穷大

11.CSS 盒子模型

页面布局三大核心：盒子模型、浮动、定位
盒子模型组成：border边框、content内容、padding内边距、margin外边距

  1. 边框（border）

    CSS边框属性允许你指定一个元素边框的样式和颜色

    语法：
    border: border-width || border-style || border-color
    简写：
    border: 1px solid/dashed/dotted red; 没有顺序
    边框分开写法：border-top: 1px dashed pink;

    样式 属性 值
    边框宽度 border-width 1px
    边框样式 border-style solid/dashed/dotted
    边框颜色 border-color pink
    边框合并 border-collapse collapse

  2. 内边距（padding）

    CSS内边距属性可以有一到四个值（顺时针）

    padding会撑开盒子大小，所以去掉盒子本身的的width/height属性

    语法：
    padding-top || padding-bottom || padding-left || padding-right
    简写：
    padding: 1px 5px 10px 20px;

    值的个数 表达意思
    padding: 5px; 代表上下左右都有5px内边距
    padding: 5px 10px; 代表上下内边距5px，左右内边距是10px
    padding: 5px 10px 20px; 代表上内边距5px，左右内边距是10px，下内边距20px
    padding: 5px 10px 20px 30px; 代表上内边距5px，右内边距是10px，下内边距20px，左内边距是30px

  3. 外边距（margin）
    3.1 基础

    margin属性用于设置外边距，即控制盒子和盒子之间的距离

    语法：
    margin-top || margin-bottom || margin-left || margin-right
    简写：
    margin: 1px 5px 10px 20px;

    值的个数 表达意思
    margin: 5px; 代表上下左右都有5px外边距
    margin: 5px 10px; 代表上下外边距5px，左右外边距是10px
    margin: 5px 10px 20px; 代表上外边距5px，左右外边距是10px，下外边距20px
    margin: 5px 10px 20px 30px; 代表上外边距5px，右外边距是10px，下外边距20px，左外边距是30px
    3.2 块级盒子水平居中

    外边距可以让块级盒子水平居中，但必须满足两个条件

    • 盒子必须指定了宽度（width）
    • 盒子左右的外边距都设置为auto

    语法：.header {width:960px; margin:0 auto}

    以上方法是让块级元素水平居中
    行内元素或行内块元素水平居中：给其父元素添加以下代码

    语法：text-aling: center

    3.3 外边距合并

    使用margin定义块元素的垂直外边距（上下外边距）时，可能会出现外边距的合并

    1. 相邻块元素垂直外边距的合并（取两个值中的较大者）

      解决方案：尽量只给一个盒子添加margin值

    2. 嵌套块元素垂直外边距的塌陷（取两个值中的较大者） ：

      解决方案：

      • 1.可以给父元素定义上边框（恩断义绝border框）
      • 2.可以给父元素定义上内边框（恩断义绝padding框）
      • 3.可以给父元素添加overflow: hidden

  4. 清除内外边距

    网页元素很多都带有默认的内外边距，不同的浏览器默认的内外边距不同；因此在布局前需要清除网页元素的内外边距 

    * {
 padding: 0;  /*清除内边距*/
 margin: 0;  /*清除外边距*/
}

    注：行内元素为了照顾兼容性，尽量只设置左右内外边距，不要设置上下内外边距，但是转换成块级和行内块元素就可以了
