### 4.5 图像标签和路径
```
```
属性 属性值 说明
—- :—- :—-
src 图片路径 必须属性
alt 文本 替换文本。图像不能显示的文字
title 文本 提示文本。鼠标放到图像上，显示的文字

+ 相对路径

相对路径：以引用文件所在位置为参考基础，而建立出的目录路径（图片相对于HTML页面的位置）

|相对路径分类|符号|说明|
|----|:----|:----|
|同一级路径||图像文件位于HTML文件同一级 如```<img src="haha.jpg" />```|
|下一级路径|/|图像文件位于HTML文件下一级 如```<img src="images/haha.jpg" />```|
|上一级路径|../|图像文件位于HTML文件上一级 如```<img src="../../haha.jpg" />```|
  • 绝对路径

    绝对路径：指目录下的绝对位置，直接到达目标位置，通常是从盘符开始的路径

    <img src="C:\Users\L\Pictures\haha.jpg" />
<img src="https://www.r2coding.com/r2_static/img/haha.png" />

    4.6 超链接标签

    1. 外部链接
        <a href="https://zhangxuelee.github.io/" target=_self>个人网站</a>
      属性 作用
      href 用于指定链接目标的url地址，（必须属性）
      target 用于指定链接页面的打开方式，其中_self为默认值，_blank为在新窗口中打开方式
    2. 内部链接

      内部链接：网站内部页面之间的相互链接，直接链接内部页面名称即可

       <a href="index.html" >主页</a>
    3. 空链接

      空链接：如果当时没有确定链接目标时使用

        <a href="#" >主页</a>
    4. 下载链接

      下载链接：如果href里面地址是一个文件或者压缩包，会下载这个文件

         <a href="favicon.zip" >下载文件</a>
    5. 网页元素链接

      网页元素链接：在网页中的各种网页元素，如文本、图像、表格、音频、视频等都可以添加超链接

        <a href="https://zhangxuelee.github.io/"><img src="favicon.png" /></a>
    6. 锚点链接

      锚点链接：点击链接可快速定位到页面中指定位置。
      设置href属性值为”#名字”，在目标位置标签中添加id属性，赋值为刚才的名字

        <a href="#live" />个人生活</a>
  <h4 id="live">个人生活</h4>

      4.7 表格标签

      表格用于展示数据

      1.表格的相关标签:

      <tr>标签用于定义行
<td>标签用于定义单元格
<th>标签用于表示HTML表格的表头部分
<thead>标签是表格的头部区域
<tbody>标签是表格的主体区域
注：表头单元格里面的文本内容会加粗居中显示

      2.表格的书写格式:

      <table align="center" border="1" >
<thead>
<tr>
  <th>姓名</th>
  <th>性别</th>
</tr>
</thead>
<tbody>
<tr>
  <td>张三</td>
  <td>男</td>
  ...
</tr>
...
</tbody>
</table>

      3.表格的相关属性:

      属性名 属性值 描述
      align left、center、right 规定表格相对周围元素的对齐方式
      border 1或”” 规定表格单元是否拥有边框，默认为””,表示没有边框
      cellpadding 像素值 规定单元格边沿与其内容之间的空白，默认1像素
      cellspacing 像素值 规定单元格之间的空白，默认2像素
      width 像素值或百分比 规定表格的宽度
      #### 4.合并单元格
      ```
      跨行合并:rowspan=”合并单元格的个数”,写在最上侧单元格
      跨列合并:colspan=”合并单元格的个数”,写在最左侧单元格
      ```
      注:最后需要删除多余的单元格

4.8 列表标签

列表标签用于布局

1.无序列表

<ul></ul>中只能嵌套<li></li>
<li></li>中可以放任何标签
<ul>
  <li>列表项1</li>
  <li>列表项2</li>
  <li>列表项3</li>
  ...
</ul>

2.有序列表

<ol></ol>中只能嵌套<li></li>
<li></li>中可以放任何标签
<ol>
  <li>列表项1</li>
  <li>列表项2</li>
  <li>列表项3</li>
  ...
</ol>

3.自定义列表

自定义列表常用于对术语或名词进行解释和描述,列表项前无序号

<dl></dl>中只能嵌套<dt><dd>
<dt><dd>中可以放任何标签
<dl>
  <dt>名词1</dt>
  <dd>名词1解释1</dd>
  <dd>名词1解释2</dd>
  ...
</dl>

4.9 表单标签

表单用于收集用户信息

1.表单域

<form>标签用于定义表单域,<form>会将它范围内的表单元素信息提交给服务器
<form action="url地址" method="提交方式" name="表单域名称">
  各种表单元素控件
</form>

2.表单控件

  • input输入表单元素

    <input>标签用于收集用户信息

    <label>标签用于扩大按钮选择范围，其中for和<input>标签中的id值必须一致

    <input type="radio" id="nan"name="sex" value="男" checked="checked"><label for="nam">男</label>
    <input type="text" name="username" value="请输入用户名" maxlength="6">

    type属性:

    属性值 描述
    button 按钮
    checkbox 复选框，name值必须一致，设置checked属性用于默认选中此按钮
    password 密码框
    text 文本框，设置maxlength属性用于限制字符长度
    radio 单选按钮，name值必须一致，设置checked属性用于默认选中此按钮
    submit 提交按钮，value=”提交”
    reset 重置按钮，value=”重新填写”
    file 文件域，用于上传文件

  • select下拉表单元素

    <select>标签用于定义下拉列表，selected=”selected”属性为默认选中

    <form>
  <select>
    <option>选项1</option>
    <option>选项2</option>
    <option selected="selected">选项3</option>
    ···
  </select>
</form>

    注：<select>标签内最少有一对<option>标签

  • textarea文本域元素

    <textarea>标签用于多行文本输入的控件，常见于留言板、评论

    <form>
留言板：
  <textarea rows="3" cols="20">
    最多不超过500字
  </textarea>
</form>
    属性值 描述
    rows 显示的行数，一般用css实现
    cols 每行中的字符数，一般用css实现
